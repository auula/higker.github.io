在我们开发调试程序的时候我们的一些程序部署在服务器上运行，但是很多配置文件，这样我们修改起来比较麻烦，使用我们可以通过vscode服务器版本进行开发。

开始安装

安装可以通过这个https://github.com/cdr/code-server仓库上面的文件来。

根据自己对应的系统版本安装

Debian, Ubuntu

curl -sSOL https://github.com/cdr/code-server/releases/download/v3.3.1/code-server_3.3.1_amd64.deb
sudo dpkg -i code-server_3.3.1_amd64.deb
systemctl --user enable --now code-server
# Now visit http://127.0.0.1:8080. Your password is in ~/.config/code-server/config.yaml

Fedora, Red Hat, SUSE

curl -sSOL https://github.com/cdr/code-server/releases/download/v3.3.1/code-server-3.3.1-amd64.rpm
sudo yum install -y code-server-3.3.1-amd64.rpm
systemctl --user enable --now code-server
# Now visit http://127.0.0.1:8080. Your password is in ~/.config/code-server/config.yaml

Arch Linux

# Installs code-server from the AUR using yay.
yay -S code-server
systemctl --user enable --now code-server
# Now visit http://127.0.0.1:8080. Your password is in ~/.config/code-server/config.yaml
# Installs code-server from the AUR with plain makepkg.
git clone https://aur.archlinux.org/code-server.git
cd code-server
makepkg -si
systemctl --user enable --now code-server

默认访问地址: http://127.0.0.1:8080. 默认密码配置:~/.config/code-server/config.yaml

Firewall wall setting port

–permanent 添加这个参数防火墙永久生效.

默认端口是 8080.

$: firewall-cmd --zone=public --add-port=8080/tcp --permanent

# 刷新配置

$: firewall-cmd --reload

Edit config file

使用 vi 打开配置文件~/.config/code-server/config.yaml

$: vi  ~/.config/code-server/config.yaml

使用键盘i 键进入insert模式

 bind-addr: 127.0.0.1:8080       
 auth: password
 password: 706039b9fa******b847cec0c
 cert: false
 ~
 ~
 -- INSERT --

bind-addr 修改为 0.0.0.0:8080 (这样就可以远程访问)
password 修改为你自定义密码

按键盘上的esc 键输入: wq 保存退出文件

启动服务器

$: /usr/bin/code-server

output:

[root@vultr ~]# /usr/bin/code-server
info  Wrote default config file to ~/.config/code-server/config.yaml
info  Using config file ~/.config/code-server/config.yaml
info  Using user-data-dir ~/.local/share/code-server
info  code-server 3.3.1 6f1309795e1cb930edba68cdc7c3dcaa01da0ab3
info  HTTP server listening on http://127.0.0.1:8080
info      - Using password from ~/.config/code-server/config.yaml
info      - To disable use `--auth none`
info    - Not serving HTTPS

不需要密码可以添加 -auth none 参数

or start by designated user

$:  systemctl --user enable --now code-server

now visit http://your.server.address:8080. Good luck~
